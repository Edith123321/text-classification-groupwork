# Embeddings

This folder contains Python modules for generating text embeddings using different techniques.

## Files

- **tfidf.py**: TF-IDF (Term Frequency-Inverse Document Frequency) embedding generation
- **word2vec_skipgram.py**: Word2Vec Skip-gram model for word embeddings
- **word2vec_cbow.py**: Word2Vec Continuous Bag of Words (CBOW) model for word embeddings
