# Notebooks

This folder contains Jupyter notebooks for training and evaluating different machine learning models.

## Files

- **01_logistic_regression.ipynb**: Logistic regression model training and evaluation
- **02_rnn.ipynb**: Recurrent Neural Network (RNN) model training and evaluation
- **03_lstm.ipynb**: Long Short-Term Memory (LSTM) model training and evaluation
- **04_gru.ipynb**: Gated Recurrent Unit (GRU) model training and evaluation

Each notebook includes data loading, model training, evaluation metrics, and visualization of results.
